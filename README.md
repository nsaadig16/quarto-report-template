# Template Quarto report

Template for a Quarto report for the University of Lleida.

## Requirements

Make sure to have [Quarto](https://quarto.org/docs/get-started/),  [uv](https://docs.astral.sh/uv/getting-started/installation/)  and Make installed on your system.

## Commands

- `make preview`: Shows a preview of the report on your browser.
- `make render`: Renders the report to a PDF file on the `output/` folder.
- `make clean`: Cleans up the project.
